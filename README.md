# Excel: Mental Health & Social Media Impact

This repository showcases an Excel dashboard project built using a [Kaggle dataset](https://www.kaggle.com/datasets/prince7489/mental-health-and-social-media-balance-dataset). It includes an interactive dashboard that analyzes the relationship between social media use and mental health, using pivot tables, slicers, and data visualizations.

## What’s inside
An interactive Excel dashboard with slicers to filter by:
- Gender
- Age group
- Daily screen time category

Key visuals include:
- Screen time vs Mental well-being indicators (happiness, sleep quality, stress)
- Average daily time spent by social media platform
- Age-based social media engagement patterns
- Screen time vs Weekly exercise participation

## Tools & techniques (Excel)
- Data cleaning and preparation in Excel
- PivotTables / PivotCharts
- Slicers for interactivity
- Calculated fields / summary tables
- Correlation-style exploration (in a separate pivot table sheet)

## How to use
1. Download `Mental_Health_and_Social_Media_Balance_Dataset_done.xlsx`
2. Open in Excel and click **Enable Editing** (and **Enable Content** if prompted)
3. Use the slicers on the left to explore different segments
